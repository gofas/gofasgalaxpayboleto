# Gofas GalaxPay Boleto

Módulo de gateway de pagamento para WHMCS que integra geração de boletos bancários via API GalaxPay. Desenvolvido pela Gofas Software.

## Funcionalidades

- Geração de boletos via API GalaxPay
- Verificação e baixa automática

## Requisitos

- WHMCS 7.x ou superior
- PHP 8.x
- Conta GalaxPay com API habilitada (Galaxy ID e Hash)

## Instalação

1. Copiar `modules/gateways/` para o `modules/gateways/` do WHMCS
2. Ativar e informar Galaxy ID e Hash
